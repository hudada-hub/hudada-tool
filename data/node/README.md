# NPM 学习笔记

这个项目用于学习 NPM（Node Package Manager）的基础知识和进阶特性。

## 学习大纲

1. NPM 基础
   - npm init 的使用
   - package.json 文件解析
   - 基本命令介绍

2. 依赖管理
   - 安装依赖包
   - dependencies vs devDependencies
   - 版本控制和语义化版本

3. NPM 脚本
   - 创建和运行脚本
   - pre/post 钩子
   - 自定义脚本命令

4. 包管理
   - 发布包
   - 更新包
   - 删除包

5. NPM 配置
   - npm config
   - .npmrc 文件
   - 私有仓库配置
